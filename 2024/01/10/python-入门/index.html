不支持重复元素，内容无序

定义set

//字面量

变量名称 = {ele1,ele2,ele3,...}

集合的常用操作

修改

集合时无序的，不支持下表索引访问 但集合和列表一样，允许修改

添加新元素add()
myset = {"aa","bb","bb","cc"}
myset.add("dd")
print(myset)
//结果为：{"aa","bb","cc","dd"}，将重复的"bb"删除一个
移除元素remove()
myset.remove("aa")//集合本身被修改，移除了元素
print(myset)
//结果为：{"bb","cc","dd"}
随机取出一个元素pop()
ele = myset.pop()
// 更改了原集合
取出2个集合的差集difference()
set1 = {1,2,3}
set2 = {1,5,6}
set3 = set1.difference(set2)//取出集合1和集合2的差集，集合1有的而集合2没有的，生成新的集合
print(set3)
// 结果为：{2,3}
print(set1)//{1,2,3}
print(set2)//{1,5,6}
//没有改变原集合
消除差集difference_update()
set1.difference_update(set2)//在集合1内删除和集合2相同的元素，没有生成新的集合
print(set1)//{2,3}
print(set2)//{1,5,6}
//集合1被修改，集合2不变
2个集合合并union()
set3 = set1.union(set2)//将集合1和集合2合成新的集合
print(set3)//{1,2,3,5,6}
print(set1)//{1,2,3}
print(set2)//{1,5,6}
//原集合没有被修改
集合的遍历

不能通过while循环遍历，但可以通过for循环遍历

for ele in set1:
    print(ele)//1,2,3

（2）函数多种传参方式

位置参数

def user_info(name,age,gender):
    print(f"名字是{name},年龄是{age},性别是{gender}")
//调用
user_info(name='TOM',age=20,gender='男')
user_info(age=20,name='TOM',gender='男')

缺省参数

def user_info(name,age,gender='男'):
    print(f"名字是{name},年龄是{age},性别是{gender}")
//调用
user_info(name='TOM',age=20)
user_info('Rose',18,'女')

不定长参数

不定长参数也叫可变参数，用于不确定调用的时候会传递多少个参数（不传参也可以）的场景 作用：当调用函数时不确定参数个数时，可以使用不定长参数 类型：位置传递、关键字传递

位置传递
def user_info(*args):
    prinf(args)

//调用
user_info('TOM')
user_info('TOM',18)

注意：传进的所有参数都会被args变量搜集，他会根据传进参数的位置合并一个元组(tuple)，args是元组类型

关键字传递
def user_info(**kwargs):
    print(kwargs)

//调用
user_info(name='TOM',age='18',id=110)

注意：参数是“键=值”形式的情况下，所有的“键=值”都会被kwargs接收，同时会根据“键=值”组成字典dict

（4）lambda匿名函数

函数的定义中 def关键字，可以定义带有名称的函数，有名称的函数可以重复使用 lambda关键值，可以定义匿名函数（无名称），匿名函数只可以临时使用一次

语法：lambda 传入参数：函数体（一行代码）

def test_func(compute):
    result = compute(1,2)
    print(result)
//调用
test_func(lambda x,y:x + y)

4、文件操作

（1）文件的编码

推荐UTF-8 ### （2）文件的读取 #### 打开文件 > open()打开函数

open(name,mode,encoding)
f = open("python.txt","r",encoding="UTF-8")

name：是要打开的目标文件名的字符串(可以包含文件所在的具体路径) mode：设置打开文件的模式（访问模式）：只读'r'，写入'w'，追加'a' r:以只读方式打开文件。文件的指针将会放在文件的开头。这是默认模式 w:打开一个文件只用于写入。如果该文件已经存在则打开文件，并从开头开始编辑，原有的内容会被删除。如果该文件不存在，创建新的文件。 a:打开一个文件用于追加。如果该文件已经存在，新的内容将会被写入到已有的内容之后。如果该文件不存在，创建新文件进行写入。 encoding：编码格式（推荐UTF-8）

读取文件

read()函数
f.read(num)
//num表示要从文件中读取的数据的长度（单位是字节），如果没有传入num，那么就表示读取文件中所有数据。

readlines可以按照行的方式吧整个文件中的内容进行一次性读取，并且返回的是一个列表，其中每一行的数据为一个元素。

f = open("python.txt")
content = f.reanlines()
print(content)//['aa\n','bb\n']

with open("python.txt","r",encoding="UTF-8") as f

        print(f"每一行数据为{line}")

不需要人为关闭文件 f.close()

  • 举例 > 文件内容为： > hello world python yes > yes hello aaa > world python c++

  • 要求判断python出现的次数

方式1

//打开文件
f = open("python","r",encoding = "UTF-8")
content = f.read()//字符串格式
count = content.count("python")

方式2

count = 0//累计python出现的次数
//打开文件
f = open("python","r",encoding = "UTF-8")
for line in f:
    line = line.strip()//去除开头的空格和换行符
    words = line.split(" ")//以空格分隔为字典
    for word in words:
        if word == "python":
            count += 1
print(count)

（3）文件的写入

f = open("python.txt","w")
f.write("hello world")
f.flush()//内容刷新
f.close()

直接调用write，内容并未真正写入文件，而是会积攒在程序的内存中，称之为缓冲区 当调用flush的时候，内容会真正写入文件

（1）异常的捕获方式

捕获常规异常

基本语法

try:
    可能发生错误的代码

    如果出现异常执行的代码

捕获指定异常

基本语法

try:

    print(e)
  • 注：
  • （1）如果尝试执行的代码的异常类型和耀捕获的异常类型不一致，则无法捕获异常。
  • （2）一般try下方只放一行尝试执行的代码

捕获多个异常

基本语法

try:
    print(1/0)
except (NmaeError,ZeroDivisionError) as e:

  • 注：当捕获多个异常时，可以把耀捕获的异常类型的名字，放在except后，并使用元组(tuple)的方式进行书写

捕获所有异常

可以使用基本语法捕获

try:
    print(name)

异常else

else表示的是如果没有异常要执行的代码

异常的finally

finally表示的是无论是否异常都要执行的代码，例如关闭文件

try:
    f = open("python.txt","r")

    f = open("python.txt","w")
else:
    print("没有异常")
finally:
    f.close()// 有没有异常都要执行

str_util.py文件

def str_reverse(s):
    """
    接收传入字符串，将字符串反转返回
    :param s: 接收的字符串
    :return: 返回反转字符串
    """
    return s[::-1]

def substr(s,x,y):
    """
    按照下标x,y，对字符串进行切片
    :param s: 输入的字符串
    :param x: 开头
    :param y: 结尾
    :return: 切片出来的字符串
    """
    return s[x:y]

if __name__ == '__main__':
    print(str_reverse("abcd"))
    print(substr("abcd", 1, 2))

file_util.py文件

    def print_file_info(file_name):
    """
    接收传入文件的路径，
    打印文件的全部内容，
    如文件不存在则捕获异常，
    输出提示信息，
    通过finally关闭文件对象
    :param file_name:
    :return:
    """
    #打印文件全部内容
    f = None
    try:
        f = open(file_name, "r", encoding="UTF-8")
        content = f.read()
        print(content)
    except Exception as e:
        print("文件不存在")
    finally:
        if f:   #如果f没有读取内容，直接执行close也是有问题的
            f.close()

def append_to_file(file_name,data):
    """
    接收文件路径以及传入数据，将数据追加写入文件中
    :param file_name:
    :param data:
    :return:
    """
    f = open(file_name, "a", encoding="UTF-8")
    f.write(data)
    f.write("\n")
    f.close()

if __name__ == '__main__':
