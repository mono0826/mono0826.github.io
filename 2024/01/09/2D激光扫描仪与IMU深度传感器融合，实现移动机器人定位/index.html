2D激光扫描仪与IMU进行深度学习融合，实现移动机器人定位

发表于 2024-01-09 更新于 2024-01-10 分类于 深度学习SLAM 本文字数： 5.5k 阅读时长 ≈ 20 分钟

Abstract

多传感器融合在基于2D激光的机器人定位和导航中起着关键作用。尽管它取得了巨大的成功，但仍存在一些挑战，例如，在具有较大的角度旋转时会变得脆弱。在本文中，我们提出了一种基于深度学习的方法，使用二维激光和惯性测量单元定位移动机器人。开发了一种基于递归卷积神经网络 （RCNN） 的新型架构，用于融合激光和惯性数据以进行扫描到扫描位姿估计。还引入了扫描到子图优化，以优化 RCNN 估计的姿态，以提高鲁棒性和准确性。在仿真和实践中，对真实移动机器人进行了大量的实验，验证了所提出的深度传感器融合系统的有效性。

1. INTRODUCTION

激光扫描仪广泛用于机器人定位和建图[1]。然而，当面对具有挑战性的场景时，仅基于激光数据的定位算法可能不可靠[2]。提高基于激光的机器人定位和导航鲁棒性的主要技术之一是多传感器融合。

多传感器融合在机器人导航和定位领域起着关键作用，尤其是在考虑嘈杂的传感器读数的不确定性时。贝叶斯滤波器，例如卡尔曼滤波器和粒子滤波器，被设计用于融合多个信息源，以优化机器人姿态的估计[3]，[4]。在文献[5]中，卡尔曼滤波用于将基于激光的迭代最近点（ICP）配准算法与惯性测量单元（IMU）融合。在[6]，[7]中也引入了类似的想法。其他类型的传感器组合也适用于传感器融合，例如带IMU的超声波传感器[8]、带立体视觉的激光扫描[9]和带IMU的超宽带系统[10]。尽管这些方法已经取得了令人印象深刻的性能，但它们通常需要严格的时间同步和传感器之间的外部校准[11]。同时，它们缺乏学习能力，无法自动从随时间收集的大量传感器数据中受益。如何从激光和惯性数据中直接学习传感器融合以进行机器人定位仍然是一个悬而未决的问题。

激光里程计是激光机器人定位系统的主要要素之一，其精度和鲁棒性在一定程度上决定了整个系统的性能。不幸的是，当遇到具有显着角度变化的运动时，激光里程计往往不可靠。一种可能的解决方案是使用从这些具有挑战性的场景中收集的传感器数据来训练深度学习模型。然而，使用深度模型学习激光里程计的工作非常有限。

在本文中，我们提出了一种基于递归卷积神经网络（RCNN）的传感器融合框架，用于使用激光扫描仪和IMU进行机器人定位。该框架被表述为序列到序列姿态回归问题。我们的主要贡献如下：

  • 该文设计了一种基于RCNN的激光惯性位姿估计传感器融合算法。据我们所知，这是第一个基于深度学习的激光和惯性数据传感器融合。
  • 混合框架将基于优化的扫描匹配与深度学习相结合，即使在具有挑战性的场景中也能实现高精度和鲁棒性。

所提出的基于深度学习的数据融合可以融合具有不同采样率的传感器（在这项工作中，激光器和IMU分别以40Hz和100Hz运行），并且不需要传感器之间的精确时间同步或外部校准。

本文的其余部分组织如下。第二节审查了有关工作。在第三节对系统进行了概述之后，第四节介绍了所提出的基于RCNN的激光和惯性传感器融合方法。在第五节中，介绍了姿态优化，以优化从网络预测的姿态。实验结果见第六节。结论见第七节。

2 RELATE WORKS

多传感器融合旨在通过考虑不同传感器的特性来实现最佳状态估计。经典的里程计测量通常将轮式里程计与IMU相结合。在本文中，我们以激光和IMU数据为观察对象来估计机器人的姿态。

A. Robot Localization

1)Laser Odometry:近年来，激光数据的几何领域出现了越来越多的方法，如点云配准( Point Cloud Registration)、定位( location)、激光里程计( laser odometry)等。大多数流行的二维激光同步定位和建图（SLAM）解决方案，如Gmapping [12]、Hector-SLAM [13]和Google Cartographer [14]，都使用车轮里程计(wheel odometer)来提高精度。Hector-SLAM使用IMU数据作为平台，不显示滚动/俯仰运动， Cartographer和一些3D激光SLAM（LOAM [15]）在激光点配准之前使用IMU数据作为初始值，这使得配准更快、更可靠。早些时候，Zebedee [16] 使用 2D 激光扫描仪和 IMU 生成 3D SLAM。

2)Deep Learning Based Methods:现有的基于深度学习的机器人定位方法大多是基于视觉的[17]–[19]。其他机器学习技术用于闭环( loop-closure)问题，如[20]、[21]。Nicolai 等人。 [22] 利用深度学习进行基于激光的里程计估计。[23]、[24]实现了基于深度学习的激光配准，Elbaz等。 [25] 使用深度自动编码器进行点云定位。Li等人提出了一种基于深度学习的二维扫描匹配方法。 [26]，其中使用Hector SLAM取得了良好的效果。然而，深度学习对激光和惯性数据的好处在二维激光数据的几何匹配领域尚未得到充分探索。

B. Deep Learning based Multi-Sensor Fusion

由于数据融合的好处有限，Hosseinyalamdary[27]提出了基于RNN的深度卡尔曼滤波器，用于融合IMU和全球导航卫星系统（GNSS）。为了提高基于视觉的深度估计的准确性，Agresti 等人。 [28] 利用深度学习从相机和立体视觉系统中提取置信度图。基于深度学习的融合方法也用于其他领域。例如，一个网络被训练融合图像和点云以进行3D目标检测[29]，[30]。

Rambach 等人。 [31] 使用 RNN 融合了基于标准标记的视觉姿势系统和 IMU 的输出，从而消除了统计过滤的需要。克拉克等人。 [32] 通过使用深度学习网络融合视频和 IMU 数据来展示视觉惯性里程计。

3 SYSTEM OVERVIEW

在本节中，简要描述了一种基于RCNN的序列间数据融合位姿估计系统。图 1 显示了所提出的系统框架。它由两部分组成：基于 RCNN 的数据融合，用于扫描到扫描位姿估计，以及基于 ICP 的扫描到子图位姿优化。

图一 所提出的用于机器人定位的传感器融合算法的系统概述。激光和惯性数据通过循环卷积神经网络（RCNN）融合，并通过扫描到子图扫描匹配进一步优化RCNN的位姿预测。

基于RCNN的传感器融合由基于CNN的点云特征提取、基于RNN的IMU配准和基于RNN的数据融合组成。CNN旨在从两次激光扫描中提取特征以进行融合。同时，用于 IMU 配准的 RNN 在两次激光扫描之间处理 IMU 数据序列。由于激光和IMU传感器通常以完全不同的采样率运行，因此挑战之一是传感器数据流是多速率的。为了解决这个问题，我们使用额外的 RNN 来融合从激光 CNN 和 IMU RNN 中学到的特征。RNN 不仅结合了激光和 IMU 传感器的信息，还学习了机器人的运动模型。

扫描到扫描姿态估计往往是准确的，因为它是仅两次连续激光扫描之间的短时间估计。然而，累积扫描间误差的全局姿态会随着时间的推移而发生漂移。为了进一步优化RCNN的姿态预测，该文采用基于ICP的scan-to-submap算法。基于ICP的位姿优化可以减少扫描间估计的累积误差，并消除一些异常值。

A. CNN Based Feature Extraction

特征提取可以表述为非线性映射问题。自动学习有用特征表示的有效方法是深度神经网络。众所周知，卷积层可以解释为将输入转换为特征表示，可用于回归或分类问题。为了学习适合激光扫描的有用特征，开发了一种 CNN 来对一对连续的激光扫描进行特征提取。由于里程计系统需要部署在未知的环境中，因此这项工作中的CNN是提取几何特征而不是语义特征。

CNN的配置在表I中概述。网络的输入是两个激光扫描的堆栈。有六个卷积层，每个卷积层都有一个整流线性单元 （ReLU） 激活。由于在工作中采用二维激光传感器，因此网络使用一维卷积层。在最后一个卷积层之后，全局最大池化用于提取全局特征，该特征是 RNN 网络输入的一部分。

B. RNN Based Sequence Learning and Fusion

基于RNN的序列学习和融合

递归神经网络（RNN）是一种具有时间学习记忆的神经网络。它能够处理顺序数据，因为它可以利用历史信息进行当前状态预测，对序列中的依赖关系进行建模。因此，RNN非常适合涉及时间模型（运动模型）和顺序数据（激光扫描序列和IMU序列）的定位问题。例如，估计当前帧的姿态可以从前几帧中封装的信息中受益。

1)基于LSTM的序列学习(LSTM Based Sequence Learning):长短期记忆 （LSTM） 是最流行的 RNN 类型之一，因为它能够通过引入记忆门和单元来学习长期依赖关系。它的门会自动决定在训练时何时存储或丢弃数据和内存。确保 LSTM 在时刻的输入是 ，对于上一时刻的LSTM 单元，它的隐藏状态是存储单元是 。LSTM 的更新时间步长为： 表示两个向量的元素乘积， 表示sigmoid非线性函数， 是双曲正切非线性函数,表示对应的权重矩阵，表示偏置向量(bias vector)，、、、和 分别代表时间步 的输入门（input gate）、遗忘门（forget gate）、输入调节门（input modulation gate）、记忆细胞（memory cell）和输出门（output gate）。

LSTM网络将激光和IMU的联合特征表示作为输入，并学习用于定位的时序模型。我们现在下面讨论如何配准和融合激光和IMU数据。

2)IMU序列配准(IMU Sequence Registration):该系统的输出是扫描到扫描的姿态估计，它以激光帧作为我们工作的参考。由于激光和 IMU 传感器的采样率非常不同（激光的采样率约为 40 Hz，而 IMU 的采样率为 100 Hz），因此需要对它们的传感器数据进行临时配准。因此，我们使用额外的LSTM网络来学习在两次连续激光扫描之间收集的IMU数据序列的全局特征。在实践中，对应于两帧激光扫描的IMU数据的序列长度可能不是固定的。由于 RNN 结构的灵活性，不同长度的 IMU 数据可以由 LSTM 网络进行本地处理。

3)激光和IMU融合(Laser and IMU Fusion):我们使用类似于IMU RNN的结构来实现激光和IMU数据之间的融合。如图 2 所示，主 LSTM 网络的输入是分别从激光扫描的 CNN 和 IMU 数据的 LSTM 中提取的两个全局特征的串联。LSTM主网络以激光和IMU序列为输入，通过序列学习实现数据融合。它可以学习平台的复杂运动动力学，也可以学习难以手动建模的顺序依赖关系。

图2 所提出的数据融合系统中的网络架构.该网络由一维卷积层、两个LSTM层和两个全连接层组成。

5 Pose optimization using scan-to-map matching

使用扫描到地图匹配进行姿态优化

为了进一步细化姿态估计，采用几何方法对网络预测的姿态进行微调。由于定位误差主要是由扫描到扫描匹配的累积误差产生的，因此我们建议维护一个局部地图，该地图可用于局部匹配几何约束的激光扫描。基本 ICP 被设计为扫描到地图匹配的优化。

A Matching Laser Scan to A Local Map

将激光扫描与本地地图相匹配

ICP（Iterative Closest Point）微调是由RCNN估计的相对变换初始化的。ICP接受两个激光扫描和一个局部地图作为输入，并输出一个最佳匹配的变换，该变换最小化扫描中变换后的点与局部地图中对应点之间的距离。 将相对变换和局部地图的姿态结合起来，可以得到当前帧的全局姿态:

B. Local Map Updating

局部地图是一组用于匹配激光扫描的特征点。在更新本地地图的过程中，首先将新点添加到本地地图中。在 ICP 微调中从未使用或几乎不使用的点被选为新点。

其次，使用从 ICP 计算的变换来更新本地地图的姿态。视场外的点被剔除，从而降低了搜索相应点的计算复杂性。

地图点剔除可以提高计算效率，同时确保局部地图在低速或静态状态下不会过度增加。

6 Exterimental results

E Discussion

1)RCNN 提供更好的帧到帧匹配：众所周知，基于几何形状的ICP方法对初始值很敏感。当初始化错误时，它可能会受到局部最小值的影响，并具有较大的误差。相比之下，基于神经网络的方法通过提取扫描的全局特征来匹配一对扫描，这在某些情况下可能会提高鲁棒性。图 11 给出了两个示例，展示了网络比传统 ICP 算法更鲁棒的情况。

图 11.帧到帧匹配的两个示例。顶行是示例 1，底行是示例 2。（a） 和 （e） 在匹配前显示原始扫描件;（b） 和 （f） 是基本事实;（c）和（g）是ICP的匹配结果;（d） 和 （h） 是拟议的 RCNN 的匹配结果。需要注意的是，所提出的RCNN算法比传统的ICP算法效果更好。

基于网络的方法是数据驱动的，不需要精确的几何对应关系。激光扫描仪获取离散的点云数据，其中彼此之间没有一致的一对一匹配。当找不到一致的点时，基于几何的 ICP 可能无法准确匹配点。众所周知，由于 ICP 的搜索功能，剧烈的旋转运动可能会给 ICP 算法带来问题。所提出的基于深度学习的方法的缺点是模型需要大量的数据进行训练。

2)进行 RCNN 融合至关重要：图 12 显示了没有和使用所提出的深度神经网络的实验结果。很明显，如果没有 RCNN 融合提供的初始化，scan-to-submap 方法的结果会退化。另一方面，出色的初始化值可以促进更快的 ICP 收敛，并获得更好的结果。这意味着整个系统必须拥有建议的 RCNN 架构。

7 Conclusion and future work

在本文中，我们提出了一种基于深度学习的融合框架来估计机器人姿态。它将 2D 激光扫描仪与 IMU 融合在一起，以实现里程计系统。通过仿真实验和真实移动机器人，利用二维激光扫描仪和IMU的序列数据，可以有效地估计机器人的姿态。在一些具有挑战性的情况下，例如以大角速度移动，它的性能也比传统的基于几何的方法更稳健。

在三维激光里程计领域，几何方法也存在旋转角度大的问题。因此，基于深度学习的方法仍然具有相关性。为了提高里程计的准确性，通过结合RNN和CNN来学习机器人的运动动力学应该是一个很有前途的方向。此外，我们将继续探索基于深度学习的 3D 激光里程计与几何方法相结合。
