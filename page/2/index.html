基于BIM的2D LiDAR定位在变化和动态环境中实现终身室内导航的鲁棒性

发表于 2024-01-10 分类于 深度学习SLAM 本文字数： 7.3k 阅读时长 ≈ 26 分钟

代码 论文

Abstract

​ 许多研究都依赖于事实上的标准自适应蒙特卡罗定位（AMCL）(AMCL--自适应蒙特卡洛定位 - 知乎 (zhihu.com))方法，以在从建筑信息模型（BIM模型）提取的占用栅格地图（OGM）中定位机器人。然而，大多数这些研究都假设BIM模型准确地代表了真实世界，而这很少是真实的情况。参考BIM模型与现实世界（Scan-BIM偏差）之间的差异不仅是由于家具或杂物的存在，还由于在设计阶段创建的任何模型中存在的按计划和按实际建造的偏差。这些Scan-BIM偏差可能会严重影响AMCL的准确性。本文提出了一种开源方法，可以从BIM模型中生成适当的姿态图（Pose Graph）地图，以实现在变化和动态环境中的稳健的2D-LiDAR定位。首先，2D OGMs是从复杂的BIM模型自动生成的。这些OGM仅代表结构元素，允许室内自主机器人导航。然后，一种有效的技术将这些2D OGM转换为基于姿态图的地图，从而实现更精确的机器人姿态跟踪。最后，我们利用不同的地图表示结合最先进的算法，实现准确、鲁棒的定位。此外，我们在三个模拟场景中提供了各种最先进的定位算法的定量比较，这些场景具有不同程度的Scan-BIM偏差和动态代理。更准确地说，我们比较了两种粒子滤波（PF）算法：自适应蒙特卡罗定位（AMCL）和通用蒙特卡罗定位（GMCL）；以及两种基于图形的定位（GBL）方法：Google的Cartographer和SLAM工具箱，解决全局定位和姿态跟踪问题。我们发现在一个真实的办公环境中（在中等级别的Scan-BIM偏差下），AMCL的平移RMSE增加了四倍（从空环境中的8.5厘米增加到真实环境中的33.7厘米）。相反，基于位姿图的算法表现出与粒子滤波（PF）算法相比的优越性，在实际环境中实现了7.2厘米的均方根误差（RMSE）。大量实验表明，所提出的方法在变化和动态环境中，无论是使用设计好的BIM模型还是稀疏OGM，都能够实现稳健的定位，比传统的AMCL在精度和稳健性方面表现更好。

基于正态分布变换的粒子群求解扫描匹配问题的粒子群优化

发表于 2024-01-10 分类于 传统SLAM 本文字数： 14k 阅读时长 ≈ 50 分钟

代码 论文

Abstract

​ 该文提出一种进化扫描匹配方法，用于解决同步定位与映射（SLAM）中的优化问题。在这个方向上已经投入了大量的文献，然而，大多数提出的方法在优化过程方面缺乏快速收敛和简单性，这将直接影响环境地图的准确性和估计的姿态。这是一条始终活跃的研究路线，为这个问题提供了各种解决方案。在众多的SLAM方法中，正态分布变换方法（NDT）表现出了很高的性能，目前已有大量文献发表，许多研究证明了其比其他方法的有效性。然而，很少有工作有兴趣解决优化问题。所提出的解决方案基于使用粒子群优化（PSO）的NDT扫描匹配，被称为NDT-PSO。主要贡献在于解决基于PSO和迭代NDT图的位姿估计问题。NDT-PSO方法的性能已在静态和动态环境中对类似汽车的移动机器人进行的实际实验中得到证明。NDT-PSO针对不同的群体大小进行了测试，结果表明，70个颗粒足以找到最佳颗粒，同时即使在闭环时也能避免局部最小值。该算法也适用于实时应用，70 个粒子和 70 次优化过程的平均运行时间为145ms 。可以使用更少的粒子和迭代来进一步降低此值。还评估了所提方法的准确性与机器人操作系统社区中广泛使用的其他SLAM方法的准确性，结果表明NDT-PSO优于这些算法。

Abstract

多传感器融合在基于2D激光的机器人定位和导航中起着关键作用。尽管它取得了巨大的成功，但仍存在一些挑战，例如，在具有较大的角度旋转时会变得脆弱。在本文中，我们提出了一种基于深度学习的方法，使用二维激光和惯性测量单元定位移动机器人。开发了一种基于递归卷积神经网络 （RCNN） 的新型架构，用于融合激光和惯性数据以进行扫描到扫描位姿估计。还引入了扫描到子图优化，以优化 RCNN 估计的姿态，以提高鲁棒性和准确性。在仿真和实践中，对真实移动机器人进行了大量的实验，验证了所提出的深度传感器融合系统的有效性。

LCDNet：LiDAR SLAM的深度闭环检测和点云配准

发表于 2024-01-09 更新于 2024-01-10 分类于 深度学习SLAM 本文字数： 5.2k 阅读时长 ≈ 19 分钟

代码 论文

Abstract

闭环检测是同步定位和映射 （SLAM） 系统的重要组成部分，可减少随时间累积的漂移。多年来，已经提出了几种深度学习方法来完成这项任务;然而，与手工制作的技术相比，它们的性能一直不尽如人意，尤其是在处理回环检测时。在本文中，我们介绍了一种新的闭环检测网络（LCDNet），该网络通过同时识别先前访问过的地方并估计当前扫描和地图之间的六个自由度相对变换来有效地检测光探测和测距（LiDAR）点云中的闭环。LCDNet由一个共享编码器(shared encoder)、一个提取全局描述符的位置识别头(place recognition head)和一个估计两个点云之间转换的相对姿态头(relative pose head)组成。我们引入了一种基于不平衡最优传输理论的新型相对姿态头，我们以可微分的方式实现该理论，以允许端到端训练(end-to-end training)。LCDNet在多个真实世界自动驾驶数据集上的广泛评估表明，我们的方法大大优于最先进的闭环检测和点云配准技术，尤其是在处理闭环检测时。此外，我们将我们提出的闭环检测方法集成到LiDAR SLAM库中，以提供完整的建图系统，并演示在看不见的城市中使用不同传感器设置的泛化能力。

DeepPointMap：利用统一的神经描述符推进LiDAR SLAM

发表于 2024-01-08 更新于 2024-01-10 分类于 深度学习SLAM 本文字数： 7.6k 阅读时长 ≈ 27 分钟

Abstract

点云在各个领域都显示出显著的潜力，包括同时定位和建图（SLAM）。然而，现有的方法要么依赖于密集的点云以实现高定位精度，要么使用通用描述符来减小地图大小。不幸的是，这两个方面似乎相互冲突。为了解决这个问题，我们提出了一个统一的架构:DeepPointMap，在这两个方面都具有出色的优势。我们利用神经网络从点云中提取高度代表性和稀疏的神经描述符，实现内存高效的地图表示和准确的多尺度定位任务（例如里程计和闭环检测）。此外，我们通过将其扩展到更具挑战性的多代理协作SLAM来展示我们框架的多功能性。在这些场景中获得的有希望的结果进一步强调了我们方法的有效性和潜力。
